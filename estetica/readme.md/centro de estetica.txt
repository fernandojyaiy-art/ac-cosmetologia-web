# Web Centro de Estética

Sitio web institucional para un centro de estética dedicado al cuidado de la piel.
El objetivo del proyecto es brindar información clara sobre los servicios ofrecidos
y facilitar la reserva de turnos mediante WhatsApp.

## Objetivo del proyecto
- Tener presencia online profesional
- Mostrar servicios de estética y cuidado de la piel
- Facilitar el contacto con clientes
- Servir como base para futuras mejoras (backend y app)

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (básico)

## Funcionalidades
- Página de inicio con presentación del centro
- Sección de servicios
- Página de contacto
- Formulario de consulta
- Reserva de turnos a través de WhatsApp
